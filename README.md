# Endless runner game

A simple runner game with two types of obstacles (block and stun), power-ups and coins. There is an AI chaser than knows the optimum path through the map and once it catches you, you lose. Tracks your high score and lets you save and replay maps, lastly you can view your statistics at the end of the game. This game, built with winforms, increases in difficult over time ensuring you are always challenged.

## Features

- Power-ups
- Coins
- High score
- Map saving and loading
- AI chaser
- Random map generation
- 2 types of obstacles
- 2 different speeds for the player

## Installation

need visual studio 2022 with the .NET framework

## Controls

- "w" to change the speed of the player
- "a" and "d" to move left and right respectively
